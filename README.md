# WP-JSON Enumerator
A simple python script that enumerates all possible URLs from a Wordpress JSON API

# Requirements
Python 3.x+ is *required*

# Install
```
pip install -r requirements
```

# Usage
```
python3 wpjsonenumerator http://www.example.com
```
